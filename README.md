### CareGap

**Medical-desert planning app for district-level healthcare deployment.**

[![Repo](https://img.shields.io/badge/caregap--medical--desert--map-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/stevenybuilder/caregap-medical-desert-map)
[![Cloud Run](https://img.shields.io/badge/live-Cloud%20Run-4285F4?style=for-the-badge&logo=googlecloud&logoColor=white)](https://caregap-app-31043195041.us-central1.run.app/)
[![Devpost](https://img.shields.io/badge/devpost-project-003E54?style=for-the-badge&logo=devpost&logoColor=white)](https://devpost.com/software/databricks-copilot-medical-desert-map)

</div>

CareGap ranks likely medical deserts, explains confidence, surfaces provider evidence, and recommends concrete deployment or verification actions. The project combines lakehouse-style data pipelines with statistical trust layers designed for planner workflows.

### Sentinel

**Self-healing runtime security for autonomous AI agents.**

[![Repo](https://img.shields.io/badge/sentinel--agent--security-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/stevenybuilder/sentinel-agent-security)
[![Live Demo](https://img.shields.io/badge/demo-sentinelapp--delta.vercel.app-10B981?style=for-the-badge&logo=vercel&logoColor=white)](https://sentinelapp-delta.vercel.app)

</div>

Sentinel treats the agent as the threat model. Independent investigators verify payment-agent claims against ground truth, a deterministic safety gate blocks risky actions in under a millisecond, and confirmed attacks generate new Python scoring rules that are validated and hot-deployed into the gate.

<table><tbody>
<tr><td><b>Core loop</b></td><td>Supervisor agent, payment agent, risk/compliance/forensics investigators, verdict board, deterministic gate</td></tr>
<tr><td><b>Learning system</b></td><td>Prediction errors become generated scoring functions with AST validation and clean/attack regression checks</td></tr>
<tr><td><b>Stack</b></td><td>Python, FastAPI, React, Vite, XYFlow, Aerospike, RestrictedPython, Auth0, Docker</td></tr>
</tbody></table>

### AI Meeting Autopilot

**Multimodal meeting agent that hears, sees, understands, and acts.**

[![Repo](https://img.shields.io/badge/ai--meeting--autopilot-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/stevenybuilder/ai-meeting-autopilot)
[![Cloud Run](https://img.shields.io/badge/live-Cloud%20Run-4285F4?style=for-the-badge&logo=googlecloud&logoColor=white)](https://meeting-agent-ynzg64zhoa-uc.a.run.app/)

</div>

This agent listens to live meetings, extracts commitments, checks sentiment, and executes agreed-upon actions across Slack, Calendar, Gmail, documents, and BigQuery reports. Sessions are isolated with WebSockets and async background dispatch so actions can fire while the conversation continues.

<table><tbody>
<tr><td><b>Pipeline</b></td><td>Browser audio, streaming STT, Gemini understanding, sentiment gate, async action dispatch</td></tr>
<tr><td><b>Actions</b></td><td>Slack updates, Calendar events, document revisions, Gmail summaries, BigQuery report generation</td></tr>
<tr><td><b>Stack</b></td><td>Python, FastAPI, WebSockets, Gemini, Google Cloud STT/Vision, BigQuery, Slack, Cloud Run</td></tr>
</tbody></table>

### Together

**Live translation and memory generation for cross-language presence.**

[![Repo](https://img.shields.io/badge/together--presence--agent-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/stevenybuilder/together-presence-agent)

</div>

Together is a real-time conversation agent for people separated by distance and language. It translates speech bidirectionally, narrates the other person's environment on request, then turns call moments into a generated storybook and memory video.

<table><tbody>
<tr><td><b>Live mode</b></td><td>Gemini Live voice translation, real-time captions, Vision scene analysis, WebSocket rooms</td></tr>
<tr><td><b>Memory mode</b></td><td>Storybook generation, image stylization, Veo memory videos, transcript-driven keepsakes</td></tr>
<tr><td><b>Stack</b></td><td>Python, FastAPI, Gemini Live, Google Cloud Vision, Veo, ElevenLabs fallback, Cloud Run</td></tr>
</tbody></table>
